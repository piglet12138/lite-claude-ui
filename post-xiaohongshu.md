【标题】
不用订阅不用翻墙，直接体验 Claude AI 的 Agent 能力

【正文】
之前一直想用 Claude.ai 的 Agent 功能，但要 $20/月的订阅，而且国内还不好访问。

后来自己写了一个开源版本，部署在新加坡的服务器上，现在免费开放给大家用。

能做什么：
- 联网搜索：问它任何时事问题，它会自己搜索、读全文、再回答
- 写代码：让它用 Python 做数据分析、画图表，直接出结果
- 写文档：技术博客、报告、白皮书，右侧实时预览，一键下载 Word
- 会追问：需求不明确时它会先反问你，确认清楚了再动手

跟 ChatGPT 最大的区别是，它不只是聊天。你给它一个任务，它自己规划步骤——该搜就搜，该写代码就写代码，该生成文档就生成文档。整个过程不需要你一步步指挥。
